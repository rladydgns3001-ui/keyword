키워드 평가

사고 프로세스

내보내기

📋 JSON 다운로드

전체 데이터를 JSON 형식으로 다운로드합니다.

📎 클립보드 복사

키워드 목록을 클립보드에 복사합니다.

⚙️ API 설정

Firebase Functions URL

Firebase Functions가 배포된 URL을 입력하세요.

네이버 개발자센터 API

developers.naver.com에서 애플리케이션을 등록하고 Client ID/Secret을 발급받으세요.

네이버 검색광고 API

searchad.naver.com에서 광고 계정을 만들고 API 라이선스를 발급받으세요.

⚠️ API 키는 브라우저의 localStorage에 저장됩니다.

💡 API 키 없이도 규칙 기반 키워드 생성은 사용할 수 있습니다.
