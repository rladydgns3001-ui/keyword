# 키워드 품질 개선 진행 상황

## 완료된 작업 (v5 — 2026-02-24)

### 1. 의미적 중복 제거
- `isSemanticallyDuplicate()` 함수 추가
- 공백 제거 포함 관계, 연도 접두사 제거, 핵심어(첫 단어) 비교
- "대출금리 계산기" ≈ "금리계산기" → 1개만 추천

### 2. 시즌/시한부 키워드 페널티
- `SEASONAL_PENALTY_PATTERNS` 3단계 (강-30/중-20/약-10)
- "설연휴 병원 진료" durability 80→50 하락

### 3. L2 확장 강화 + 롱테일 점수 개편
- L2 접미사 2→4개, 결과 5→7개
- 롱테일: 1w=10, 2w=35, 3w=70, 4w=90 (3단어부터 급상승)

### 4. 카테고리 제한 조정
- MAX_PER_CAT: 2→삭제→4 (완전 해제 시 금융 도배 문제 발견)

### 5. 저수익 검색어 필터
- INTENT_SCORE_MAP에 나무위키(15), 주가(15), 순위(20) 등 추가
- intentScore < 40 추천 제외
- intent 점수 로직 수정: 저수익 접미사 매칭 시 기본값(50) 아래로 내려감

### 6. 추가 최적화
- 1단어 키워드 추천 제외
- 뉴스 전문용어 블랙리스트 (예대금리차, 포용금융 등)
- intentScore 미매칭 시 -5점 감점
- 주가/시세 류 저수익 접미사 추가

### 7. 시드 쿼리 확장 + 시즌 반영
- 출산/육아, 청년 복지, 교육 시드 추가
- `getSeasonalSeeds()`: 월별 자동 시드 (1-3월: 교복비/입학, 6-8월: 냉방비 등)

### 8. 토픽 추출 품질 개선
- `GENERIC_TOPICS`: 범용 1단어("복지","건강","게임" 등) 토픽 차단
- 자동완성 폴백 시 제네릭 단어 교체 방지
- 문장 조각 필터 강화 (동사 어미, 느낌표, 5단어 이상)
- 카테고리 오분류 방지 (토픽 vs 뉴스 제목 카테고리 불일치 시 스킵)
- 토픽 중복: 글자 포함 관계 추가 ("문화복지카드" ⊂ "청년문화복지카드")

## 현재 성과

## 남은 이슈 (내일 작업)

1. **뉴스 전문용어 + 템플릿 조합 문제**: "이자 장사 가입방법", "예대 금리차 가입방법" → 말이 안 되는 키워드. 토픽이 뉴스 제목 용어일 때 템플릿 접미사 붙이기 차단 필요
2. **호출마다 결과 변동**: 뉴스가 실시간이라 어떤 호출은 S급 3개, 어떤 호출은 0개. 캐싱 또는 최근 N회 결과 병합 고려

5. **건강 카테고리 오분류**: "집값 불안해서" → 건강으로 분류되는 문제
